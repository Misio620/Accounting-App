# 🚀 個人記帳應用程式 - 快速啟動指南

## 步驟 1: 檢查開發環境

```bash
# 執行環境檢查
python setup_check.py
```

如果出現問題，按照提示進行修正。

## 步驟 2: 測試基本功能

```bash
# 測試資料庫功能
python test_basic.py
```

## 步驟 3: 安裝可選套件（推薦）

```bash
# 圖表功能
pip install matplotlib

# Excel 匯出功能
pip install openpyxl

# 或一次安裝全部
pip install -r requirements.txt
```

## 步驟 4: 啟動應用程式

```bash
# 啟動主程式
python main.py
```

## 🎯 第一次使用指南

### 1. 新增第一筆記錄
- 點擊「新增交易」或按 `Ctrl+N`
- 選擇日期、類型（收入/支出）、分類
- 輸入金額和備註
- 點擊確定

### 2. 查看統計
- 底部統計區域顯示本月收支
- 報表區域提供圖表分析

### 3. 篩選和搜尋
- 點擊「進階篩選」展開篩選選項
- 可依日期、類型、分類、關鍵字篩選
- 使用快速篩選按鈕（本月、本週、今日）

### 4. 匯出資料
- 選擇要匯出的資料（可先篩選）
- 點擊「匯出 CSV」或「匯出 Excel」
- 選擇儲存位置

## 🎮 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl+N` | 新增交易 |
| `Ctrl+E` | 編輯交易 |
| `Del` | 刪除交易 |
| `F5` | 重新整理 |
| `Ctrl+S` | 匯出 CSV |

## 📂 檔案說明

- `accounting.db` - 資料庫檔案（自動產生）
- `backup/` - 備份資料夾（建議定期備份）
- `*.py` - 程式檔案（使用 Git 管理）

## 🔧 常見問題

### Q: 程式無法啟動？
A: 執行 `python setup_check.py` 檢查環境

### Q: 圖表無法顯示？
A: 執行 `pip install matplotlib`

### Q: Excel 匯出失敗？
A: 執行 `pip install openpyxl`

### Q: 資料不見了？
A: 檢查 `accounting.db` 檔案是否存在，查看 `backup/` 資料夾
